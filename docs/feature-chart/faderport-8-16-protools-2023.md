#

### Faderport 8/16 - Pro Tools 2023+

V-Control Pro supercharges the Faderport 8/16 for pro features unavailable with standard HUI mode.

=== "Bank Modes"

    | `M`         |  | Track mute |
    | `S`         |  | Track solo |

=== "Track Browser"

    The Track Browser lets you see all tracks and jump to any track.

    | Track Browser Mode |  |  |
    |---|---|---|
    | `Track` |  | Toggle Track Browser Mode on and off |
    | `Display` |  | Six rows of 7-Character track names.|
    | `Wheel` |  | Scroll through the track list by row |
    | `Select` |  | Bank to the highlighted row track |
    |          |  | Color indicates Pro Tools track color  |

=== "Locations"
    
    Use Locations Mode to browse, recall, and create memory locations.

    | Locations Mode |  |  |
    |---|---|---|
    | `Marker` |  | Toggle Locations Mode on/off   |
    | `Display` |  | Six rows of 7-Character location names. Highlighted for browser row.|
    | `Wheel` |  | Scroll through memory locations by row |
    | `Select` |  | Recall the memory location in the highlighted row |
    |        |  | The memory location is the home location for Hot Locations |
    | `Wheel Push` |  | Push to create a new memory location |
    |  | `Prev` | Cancel new memory location |
    |  | `Next` | Confirm new memory location |

=== "Hot Locations"

    Recall memory locations named to match dedicated hot buttons for custom workflows. For example, create a memory location named `VCA` that shows only VCA tracks. Press once to see just VCAs, press again to go back to your previous view.
    
    | Hot Locations |  |  |
    |---|---|---|
    | `Audio` |  | Recall AUDIO location  |
    |  | `Shift`+ | Recall INPUTS location  |
    | `VI` |  | Recall VI location  |
    |  | `Shift`+ | Recall MIDI location  |
    | `Bus` |  | Recall BUS location  |
    |  | `Shift`+ | Recall OUTPUTS location  |
    | `VCA` |  | Recall VCA location  |
    |  | `Shift`+ | Recall FX location  |
    | `All` |  | Recall ALL location  |
    |  | `Shift`+ | Recall USER location  |
    | `Master` |  | Recall MASTER location  |

=== "Plugins"

    Edit inserts A-J, using 8 Faders and Selects to adjust pages of up to 8 continuous and 8 discrete parameters. The Faderport 16 splits into an 8 track controller on the left and plugin controller on the right.

    | `EDIT PLUGINS` |  | Faderport 8 |
    |---|---|---|
    | `Edit Plugins` |  | Enable edit insert mode |
    |                |  | Press again to cycle through inserts A-J |
    | `Fader 1-8` |  | Adjust continuous plugin parameters on line 2|
    | `Select 1-8` |  | Adjust discrete plugin parameters on line 1 |
    | `Prev` |  | Change parameter page |
    | `Next` |  | Change parameter page |
    | `Bypass` |  | Toggle plugin Bypass |
    | `Macro` |  | Show/hide plugin window |
    | | `Shift`+  | Toggle plugin Compare |

    | `EDIT PLUGINS` |  | Faderport 16 |
    |---|---|---|
    | `Edit Plugins` |  | Toggle edit plugin mode on right side |
    | `Select 1-8` |  | Select track for insert editing |
    |  |  | Cycle through inserts A-J |
    | `Fader 9-16` | | Adjust continuous plugin parameters on line 2|
    | `Select 9-16` |  | Adjust discrete plugin parameters on line 1 |
    | `Prev` |  | Change parameters page |
    | `Next` |  | Change parameters page |
    | `Bypass` |  | Toggle plugin Bypass |
    | | `Shift`+  | Toggle Bypass for all plugins on track |
    | `Macro` |  | Show/hide plugin window |
    | | `Shift`+  | Toggle plugin Compare |
